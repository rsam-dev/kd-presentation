On-Page SEO

What is On-Page SEO?

Everything you do INSIDE your website to help Google understand your content.

Easy Analogy:

Google = Teacher | Your Website = Student | On-Page SEO = How well the student explains homework

Keyword Basics

  • What is a keyword? Words people type in search
  • Why search? To find info, products, services
  • Long-tail vs Short: "spa" vs "facial spa Phnom Penh"

Examples:

Short: "spa" | Long-tail: "facial treatment spa Phnom Penh" | Local: "best spa Cambodia near me"

Activity:

Write 3 keywords customers use to find your business.

Meta Tags

Meta Title

Search result headline

Formula: Keyword + Benefit + Brand

Meta Description

Description under title

Encourages clicks

Google Result Example:

Best Facial Spa Phnom Penh | Beauty Center

www.example.com

Experience luxury facial treatments. Professional spa with natural products. Book today!

Header Tags & Images

Headers like a Book:

H1 = Title (one per page) | H2 = Chapters | H3 = Sub-topics

Image SEO:

  • Google can't see images - use alt text
  • Compress for faster loading

Bad alt: "image1.jpg" | Good alt: "woman facial treatment spa Phnom Penh"

Page Speed & Mobile

Why: Google prioritizes fast, mobile-friendly sites!

Tools:

  • PageSpeed Insights
  • GTmetrix
  • Pingdom

Tips:

  • Compress images
  • Reduce scripts
  • Enable caching
  • Use CDN

Off-Page SEO

Everything OUTSIDE your website to build trust & authority.

Analogy:

"When others recommend your business, Google trusts you more." Like getting reviews from trusted friends.

Backlinks

What are Backlinks?

When another website links to yours. Example: News site features your spa with a link.

Why Important:

  • Build authority
  • Improve rankings
  • Referral traffic

Good Backlinks:

News sites, Industry blogs, .gov, .edu sites

Bad Backlinks:

Spam sites, Link farms, Unrelated sites

Web Evolution

Why Learn This?

"Understanding how the internet evolved helps you see where customers are NOW and where marketing goes NEXT."

Web 1.0 - Static Web

Era: 1990s - Early 2000s

  • Read-only - View content only
  • No interaction - No comments/shares
  • Static HTML - Rarely changed

Examples:

Early Yahoo, Company brochure sites

Web 2.0 - Social Web

Era: 2000s - Present

  • Read + Write - Users create content
  • Social media - Interaction platforms
  • Two-way - Comments, reviews

Examples:

Facebook, YouTube, TikTok, Instagram

SEO Connection: Backlinks, shares, comments = All off-page SEO!

Web 3.0 - Intelligent Web

Era: Emerging Now

  • AI-driven - Smart, personalized
  • Blockchain - Users own data
  • Privacy - User control

Examples:

ChatGPT, Blockchain apps, Gemini
